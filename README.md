# Bank Management System (Core Java)

## Description
A console-based Bank Management System developed using Core Java.
The application allows users to create accounts, deposit and withdraw money,
and check account balance.

## Features
- Account creation
- Deposit & withdrawal
- Balance inquiry
- Menu-driven interface

## Technologies Used
- Java (Core Java)
- OOP Concepts
- Java Collections
- Exception Handling

## How to Run
1. Clone the repository
2. Compile all `.java` files
3. Run `Main.java`
